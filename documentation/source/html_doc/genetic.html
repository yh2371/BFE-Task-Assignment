Genetic Algorithm¶

This section contains documentation on functions used to start the genetic algorithm, for documentation on how the individual stages of the genetic algorithm works, please refer to the Generation, Schedule, Week and Day classes.

genetic(filename, checkerlist, generationSize, numOfReproductions, mutationRate, fitnessThreshold, month=None, year=None):

Starts genetic algorithm to run for a certain number of reproductions of generations at a given size, returns the individual (schedule) with the best score out of all generations

Parameters

  • filename (string) – name of sample set file

  • checkerlist (CheckerList) – CheckerList object

  • generationSize (int) – number of schedules in the generation

  • numOfReproductions (int) – number of reproductions

  • mutationRate (float) – the probability of mutation during reproduction for all individuals

  • fitnessThreshold (float) – the minimum fitness score an individual must have to participate in reproduction (added to the mating pool)

  • month (int) – month of schedule

  • year (int) – year of schedule

Return type

Schedule object
